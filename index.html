El Norte Despejado

SEO stands for "search engine optimization" which relates to how we can improve the visibility of say a website in search engines. This also lets us target the people we want to see our website. There are a few ways to practice this optimization, and choosing the right html tags is an important way of doing so. Below I highlight the most interesting ones I found. [1]

  • Tags realted to meta information: All that stuff in the head of the document like the title and meta description helps
  • Headers: I mean these are an overview of your content and structure
  • img alt text property: it is not only used for accessiblity but lets search engines know what your images are about.

Fun with Lists

  1. Pt: mostly for print, does not rescale well
  2. proportional to line height so can be used to keep proportions
  3. font heigh for the element
  4. Percent of whole screen, can be used to make sure the whole thing is showing or only parts of it
  5. Px: similar to pt better for print
  6. percent of size for container, can be used to proportion elements to the container, say if we want text to take the whole container
